# Create JD App

Create modern SolidJS & tRPC Full-Stack TypeScript application within seconds.

## Getting Started

```bash
npm create jd-app@latest
```

## What's the Stack?

- Serverless Vercel API + tRPC (npm: solid-trpc)
- Zod
- Prisma
- TypeScript
- SolidJS & Router
- TailwindCSS
