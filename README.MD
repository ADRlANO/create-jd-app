# Create JD App

Create modern SolidJS + TRPC full stack TypeScript application within seconds.

## Getting Started

```bash
npm create jd-app@latest
```

## Features

### Server

- TRPC
- Prisma
- Express
- TypeScript

### Client

- TRPC + Types Safety
- SolidJS
- Tailwind
- Solid Router
- TypeScript
