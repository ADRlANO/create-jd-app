# Create JD App

Create modern SolidJS & tRPC Full-Stack TypeScript application within seconds.

## Getting Started

```bash
npm create jd-app@latest
```

## tRPC

- Supports V10 Syntax
- Supports V9 Syntax

## What's the Stack?

- TypeScript
- Serverless Vercel API + tRPC (solid-trpc)
- Zod
- SolidJS
- Prisma
- Solid Router
- TailwindCSS
