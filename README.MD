# Create JD App

Create modern SolidJS & tRPC full stack TypeScript application within seconds.

## Getting Started

```bash
npm create jd-app@latest
```

## What's the Stack?

- NextJS Handler + tRPC
- Zod
- Prisma
- TypeScript
- SolidJS & Router
- TailwindCSS
