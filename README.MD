# Create JD App

Create modern SolidJS / React Native & TRPC full stack TypeScript application within seconds.

## Getting Started

```bash
npm create jd-app@latest
```

## Features

### Server

- TRPC
- Zod
- Prisma
- Express
- TypeScript

### Client

- TRPC + Types Safety
- TypeScript
- Tailwind / NativeWind
- Solid Router / React Native Navigation
- SolidJS / Expo React Native
